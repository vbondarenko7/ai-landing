01Контент-пайплайн: видео → пакет публикации
Задача

YouTube-канал выпускает длинные видео. Из каждого вручную: нарезать вертикальные Reels, выбрать сильные моменты, наложить субтитры, написать заголовки, описание, теги, посты в соцсети, сделать превью. Часы рутины на одно видео — и стиль скачет от раза к разу, смотря кто делал.

Решение

Собрал сквозной AI-пайплайн. Локальная транскрибация → отбор сильных фрагментов по правилам хуков → автонарезка вертикальных Reels с субтитрами → готовый пакет публикации: заголовки, описание, теги, посты, промпт превью. Правила зашиты в систему, поэтому результат стабильный, а не «как получится».

Итог: из одного видео — серия готовых Reels + полный пакет публикации за один прогон.
02Claude Skills под голос проекта
Задача

«Промпты в чате» каждый раз дают разный результат: тон плавает, правила теряются, у каждого в команде свой стиль, новичок долго въезжает в «как мы пишем». Знание живёт в головах, а не в системе.

Решение

Написал 8 собственных Claude Skills — формализованные правила проекта: заголовки, превью, сценарии Reels, копирайтинг, драматургия. Это не разовые промпты, а воспроизводимые инструкции, которые работают как штатный сотрудник: AI выдаёт результат в голосе проекта по канону.

Итог: правила вынесены из голов в систему — один тон, один формат, повторяемо.
03Локальная замена платных подписок
Задача

Голосовой ввод на Mac (диктовка в любое поле) — это платные сервисы: Wispr Flow $180/год, MacWhisper $59. Хотелось то же самое — но без подписки, без облака и без отправки голоса на чужие серверы.

Решение

За вечер собрал локальный аналог: whisper.cpp + фоновый сервер + глобальный хоткей + индикатор в меню-баре. Внутри та же модель, что в платных сервисах, только локально и бесплатно. Пользуюсь каждый день.

Итог: $180/год → 0, и данные никуда не уходят.
04iOS-приложение без навыков программирования
Задача

Нужен виджет на локскрин iPhone, который показывает новое английское слово с пиктограммой — для пассивного изучения. Готового бесплатного решения не было, а я не программист.

Решение

С Claude Code собрал и установил на свой iPhone настоящее iOS-приложение (SwiftUI + WidgetKit): виджет на локскрин + полноценное приложение — сотни слов, уровни сложности, несколько экранов. От идеи до работающего на устройстве — без опыта в коде.

Итог: рабочее iOS-приложение, собранное оператором, а не разработчиком.
